# 🌐 Cisco Packet Tracer Networking Labs



This repository contains my hands-on networking labs completed using Cisco Packet Tracer through Cisco NetAcad training.



## 📌 Overview


These labs helped me build a strong foundation in networking concepts, including:
- LAN setup and configuration
- IP addressing and subnetting basics
- DNS, DHCP, and HTTP traffic flow
- FTP file transfer operations
- NAT and wireless router configuration
- Packet analysis using Simulation Mode
- Routing vs Unrouted network behavior



## 🧠 Skills Gained


- Network troubleshooting using ping, ipconfig, tracert
- Understanding MAC vs IP address behavior
- Client-server communication flow
- Router, switch, and end-device configuration
- Packet-level analysis (OSI model view)
- Basic enterprise network design concepts



## 🧪 Labs Included


- Connect to Web Server
- Client Interaction & HTTP/DNS Flow
- Identify MAC and IP Addresses
- FTP File Transfer Lab
- Ping Troubleshooting Lab
- NAT on Wireless Router
- Telnet & SSH Access
- DHCP Configuration
- LAN Setup & Connectivity Testing
- Wireless Router Setup
- Routed vs Unrouted Network Traffic Flow



## 🎯 Goal


To strengthen my foundation in networking and prepare for SOC Analyst (L1) and cybersecurity roles through real-world simulation practice.



## 🚀 Tools Used


- Cisco Packet Tracer
- Cisco NetAcad Labs
